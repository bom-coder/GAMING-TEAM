## 游戏简介

Go to the door 是一款跑酷冒险类游戏，在充满邪念与恶意的地图中存活下来，到达地图中的大门才能喘息片刻并挑战下一关！
按下小键盘的左右方向键移动，上方向键跳跃。

致敬和模仿LEVEL DEVIL。 

包含多个陷阱关卡，每个关卡都有独特的挑战和谜题。
包含一个彩蛋关卡，你找到了吗？
